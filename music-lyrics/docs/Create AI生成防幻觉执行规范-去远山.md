# 《去远山》AI编曲/人声生成防幻觉执行规范
文档归属项目：去远山｜民谣二创项目
配套主文档：../PRD_去远山（仿山山外民谣二创项目）.md
文档定位：约束ACE Studio / Suno / 海绵音乐AIGC模型，四层链路阻断6类高频AI幻觉，保证编曲、歌词、节奏、结构1:1对齐原版《山山外》

## 一、本项目定义6类致命AI幻觉故障清单
| 幻觉分类 | 故障具体表现 | 风险等级 | 处理规则 |
|--------|------------|--------|--------|
| 旋律漂移幻觉 | AI自主改写原曲旋律、升降调、修改转音、变更和弦走向 | P0 致命阻断 | 音频直接作废，全曲重生成 |
| 歌词音节错位幻觉 | 无视7字硬性规则，自动增减字数、拆分合并句子、出现长短句 | P0 致命阻断 | 歌词预处理后重新输入生成 |
| 配器乱入幻觉 | 擅自增加鼓、贝斯、电声、合成器、弦乐，破坏木吉他+口琴极简框架 | P1 严重回退 | 定点重绘配器轨道，禁止全曲重生成 |
| 段落结构错乱幻觉 | 打乱「主歌1-口琴间奏-主歌2-口琴间奏-尾声」固定结构，新增副歌/桥段、无限循环尾奏 | P1 严重回退 | 锁定段落参数后重生成 |
| 演唱节奏幻觉 | 私自修改句尾长音拖拍、换气点位、句间停顿时长，人声和伴奏脱节 | P2 局部修复 | 单段落局部Repaint微调，不整曲重跑 |
| BPM/调性偏移幻觉 | 自动提速/降速、切换大小调，脱离原版固定速度与调式 | P0 致命阻断 | 重置源音频与全局参数重新生成 |

2.3 ACE Studio 专属源音频约束
上传完整无杂音《山山外》原版伴奏作为 srcaudio 种子文件，包含全部完整曲式段落
Cover 模式遵循度参数：audiocoverstrength = 1.0，关闭旋律随机偏移开关

三、第二层：模型底层参数
3.1 ACE Studio 固定防幻觉参数（保存为文本 ai_param_lock_ace.txt）
usecotmetas = True
timesignature = 4/4
temperature = 0.3  # 极低创造性，上限不可超过0.4
syllable_alignment = True  # 强制音节对齐校验，拦截非7字句式
melody_freeze = True  # 核心防漂移开关，完整锁定源音频MIDI音符
单次生成片段长度 ≤ 60s  # 超长片段会分散模型注意力，诱发结构幻觉
同一段落Repaint重绘次数 ≤ 2  # 多次迭代累积失真幻觉
3.2 Suno AI 固定防幻觉参数
creativity = 0.1  # 最低创意值，抑制自主编曲改旋律
consistency = max  # 最高内容一致性权重
no_drums = True
no_electric_instruments = True
# 强制附加标签，不可删减
same melody as reference track, fixed BPM, unchanged chord progression, strictly 7 Chinese characters per line, no extra instruments, fixed song structure

四、第三层：生成后四级自动化校验流程（拦截漏网幻觉）
P0 致命校验
音节计数校验：逐行统计汉字，每行必须严格 7 字
MIDI 旋律比对：生成人声 MIDI 与原版 MIDI 做 DTW 匹配，旋律偏移＞5% 判定漂移幻觉
BPM & 调性校验：频谱工具读取速度、调号，必须与原版完全一致
P1 配器 & 结构校验
频谱筛查：音频出现鼓 / 贝斯 / 合成器频段信号，判定配器幻觉
时间戳核对：校验间奏时长、段落顺序，禁止新增 / 删减段落
P2 演唱细节校验
逐句核对尾音拖拍、句间停顿、换气点位，偏离标准则局部重绘
人工听审，无抢拍、空拍、发音错乱问题
P3 兜底语义校验
通读歌词全文，主题统一为「奔赴山野、自我和解」，无无关跑偏意象

成品文件名：过渡期_V1.0_Demo.mp3

七、底层原理总结
AI 音乐幻觉根源：模型概率自由发挥、指令语义模糊、上下文窗口过载、缺少乐理硬过滤。
针对本锁原曲、固定句式的民谣二创项目，不能依赖 AI 自主理解需求，必须通过「硬参数锁死 + 无歧义结构化输入 + 乐理自动化校验 + 标准化归档闭环」四层强约束，压缩模型自由发挥空间，全链路规避编曲、人声、文字类幻觉，保证仓库内所有 Demo 完全匹配 PRD 规定的编曲、节奏、歌词标准。
