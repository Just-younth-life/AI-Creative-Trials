# 🏥「康伴」极简适老化健康监测平台—产品需求文档
版本：v1.0
编写日期：2026-07-04
文档状态：初稿
目标用户：65岁以上老年用户及子女/照护者
产品定位：老年用户专属的极简健康监测与医疗辅助平台

## 一、文档概述
### 1.1 文档目的
本文档旨在明确定义「康伴」极简适老化健康监测平台v1.0（MVP）的产品范围、功能规格、交互要求、适老化设计标准和非功能需求，为设计、开发、测试团队提供统一的交付依据。文档同时面向产品决策者，阐明产品的市场定位、差异化策略和迭代路径。

### 1.2 适用范围
- 产品名称：康伴（极简适老化健康监测平台）
- 发布形态：移动端WebApp（响应式）
- 技术栈：HTML+CSS+JavaScript（TailwindCSS）
- 本期版本：v1.0 MVP
- 适用团队：产品、设计、前端开发、测试、运营
- 不涵盖：后端接口、原生App

### 1.3 术语与缩写
SOS紧急求助、MVP最小可行产品、适老化、单线程任务流等定义

### 1.4 参考文档
工信部适老化规范、银发经济政策、通信行业适老化标准、内部原型地址

### 2.2 国家政策驱动

极简适老、健康闭环、零认知负担、熟人急救链四大差异化

## 三、产品定位与目标
### 3.1 产品定位
面向65岁以上老人，大字体大触控单线程，聚焦血压/用药/SOS三大场景

### 3.2 目标用户画像
独居老人张建国、异地子女李婷两套完整画像

### 3.3 核心价值主张
大字号·大按钮·不绕路——让每一位长辈都能自己管好健康。

### 3.4 产品目标与成功指标
DAU、血压记录频率、用药依从、7日留存、适老化满意度量化指标
