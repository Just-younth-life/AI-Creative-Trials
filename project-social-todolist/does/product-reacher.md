# 1 核心结论摘要

1. **赛道空白判定**：国内 C 端 Todo 工具赛道分为纯工具、纯自习社交两类，无产品同时实现「极简任务底座 \+ 全链路 AI 规划复盘 \+ 可控轻公开自律社交 \+ 情侣双人私密协同」一体化能力，交叉赛道 CAGR 达 12\.2%，增长速度高于通用清单市场 7\.9%。

2. **行为学验证**：公开选择性自律可将目标完成率从 10%\-25% 提升至 65%，搭配小组问责最高可达 95%，是解决传统 Todo「打开即走、留存极差」核心解法。

3. **竞品核心短板统一**：滴答重专业无社交 AI、番茄重自习待办薄弱、垂直工具（清单自习 / 招募 / 小习惯）功能单一、极简待办无留存抓手；全部产品缺失 AI 周期壁纸复盘、情侣双人体系、分层学生会员三大独家壁垒。

4. **商业最优解**：四层分层变现（免费广告流量 \+ 学信学生 5 折 \+ 标准版会员 \+ 高端尊享溢价），兼顾学生 DAU 与职场高净值 ARPU，AI \+ 社交大幅拉升日均时长至 25min\+，D30 留存预期 45%，远超行业均值。

5. **落地条件（条件 Go）**：默认私密、社交模块化一键关闭；AI 贯穿任务全流程而非仅语音录入；MVP 手机端优先、网页 P1、平板 P2；广告仅承载拓展模块，核心任务链路零弹窗。

## 2\.2 PEST 宏观环境分析

### P 政策（合规约束）

《个人信息保护法》约束任务、打卡、用户日程云端存储；UGC 自律榜单、复盘内容需 AI 预审 \+ 人工巡检；轻私密小组社交审核成本远低于全公开广场；iOS / 安卓商店严格限制计时中途弹窗广告，高频激励广告存在下架风险。

### E 经济

18\-35 岁年轻群体线上付费习惯成熟；学生群体价格敏感，偏好低价优惠；26 \+ 职场人群愿意付费换取纯净无广告、高阶 AI 效率能力；纯广告产品 ARPU 仅 30\-70 元，分层会员模式可拉升至 135 元 \+。

### S 社会

1. 考研 / 考公、职场混合办公常态化，自我提升需求持续爆发；

2. 用户普遍多工具混用：学生 = 番茄 \+ 有个计划，职场 = 滴答 \+ 备忘录，多 APP 切换、数据割裂是核心痛点；

3. Z 世代需要搭子监督、情绪陪伴，单纯文字清单缺少行动动力；

4. 情侣缺少一体化待办工具，现有恋爱 APP 无日程规划能力，形成独立增量市场。

### T 技术

国内大模型 API 成本持续下行，AI 任务拆解、自动周 / 月复盘可低成本落地；手机锁屏小组件、跨端云同步、多端数据互通技术成熟；语音识别、自然时间解析成为行业标配。

## 2\.3 行业核心痛点

1. 纯工具类产品：无回访动机，日均使用时长 70\-112s，D30 留存仅 12%\-32%；

2. 纯自习社交产品：任务管理体系残缺，无法承载职场复杂工作事项；

3. AI 功能普遍浅层，仅支持语音新建任务，无周期复盘、个性化效率教练；

4. 商业化两极分化：滴答高价无流量广告、番茄广告泛滥差评、轻量化工具变现单薄；

5. 无产品兼顾单人自律 \+ 情侣双人两大使用场景，市场天花板受限。

## 3\.2 各竞品核心优势 \& 致命流失诱因

1. 滴答清单
优势：完整子任务、日历双向联动、成熟多端同步、办公文件挂载；
流失诱因：89 元年定价偏高，无自律社交、AI 仅基础识别，缺少周期复盘沉淀。

2. 番茄 Todo
优势：联机自习、小组时长 PK、全套白噪音，学生流量庞大；
流失诱因：激励广告频繁弹窗打断专注，待办优先级体系简陋，无职场协同 AI。

3. 清单自习室
优势：免费锁屏小组件、多人自习房间；
流失诱因：无中长期任务拆解，仅支持短时自习，月度流失严重。

4. 有个计划
优势：看板视图、艾宾浩斯复习规划；
流失诱因：无番茄专注、无社交搭子陪伴，单独使用动力不足。

5. 招募计划
优势：自建小组共同目标、进度互见；
流失诱因：无完整日程、无番茄计时，仅打卡无法承载日常工作学习。

6. 小习惯
优势：习惯连续打卡日历；
流失诱因：仅适配喝水 / 早睡微小事项，复杂项目任务不支持。

7. Todo 清单
优势：轻量化跨端同步、基础番茄；
流失诱因：无社交 PK、无深度 AI 复盘，用户用完即走。

8. 极简待办
优势：零学习成本纯白界面；
流失诱因：无 AI、无社交、无复盘，D30 留存仅 12%。

# 4 目标用户分层画像

## 4\.1 A 类：18\-26 考研 / 考公学生（核心流量人群）

行为：日均自律 APP 使用 218 分钟（QuestMobile）；
场景：宿舍、图书馆长时间自习，需要研友互相监督；
痛点：多 APP 切换，任务不会拆分，缺少月度沉淀复盘，广告过多干扰学习；
核心诉求：极简清单 \+ 番茄白噪音 \+ 搭子 PK \+ 低价学生会员。

## 4\.2 B 类：26\-35 一二线职场白领（核心利润人群）

行为：混合办公，工作事项繁杂；
场景：办公室、通勤规划日程，周报复盘；
痛点：工具功能割裂，反感广告，需要文件 / 邮件联动，无专业 AI 工作复盘；
核心诉求：纯净无广告、日历联动、高阶尊享会员。

## 4\.3 C 类：ADHD / 自由职业小众人群

行为：任务启动困难，需要陪伴式专注；
痛点：独自工作极易拖延，现有工具要么太重要么无陪伴；
诉求：低压力小组自习、轻量化任务拆解。

## 4\.4 D 类：18\-32 情侣 / 夫妻（增量蓝海人群）

场景：约会规划、共同攒钱、双人学习、纪念日管理；
痛点：恋爱 APP 无任务体系，Todo 无双人共享；
诉求：私密双人清单、共同目标、双人月度复盘。

# 5 产品五层用户体验架构

## 5\.1 战略层

### 我方产品核心价值

一套产品替代「滴答 \+ 番茄 \+ 招募计划 \+ 有个计划」，同时覆盖单人自律 \+ 情侣双人；
基础极简工具永久免费，AI \+ 轻社交为可选模块化功能，不强制打扰极简用户。

### 商业模式画布精简

|项目|内容|
|---|---|
|价值主张|极简 AI 日程 \+ 可控自律社交 \+ 情侣私密协同 \+ 周期复盘复利|
|目标用户|备考学生、职场白领、情侣双人、轻度自律人群|
|核心资源|自研任务底座、国内大模型聚合 API、轻量化社交图谱|
|盈利渠道|激励广告（免费用户）\+ 分层订阅会员（学生 / 标准 / 尊享）|
|关键业务|任务管理、AI 智能规划、私密小组 PK、双人协同、复盘报告生成|

## 5\.2 结构层

全局一级导航（默认纯净极简）：我的一天｜清单｜日历
拓展可折叠模块（用户主动开启才展示）：

1. 专注中心（番茄、白噪音、时长 PK）

2. AI 复盘（周 / 月文字、壁纸报告）

3. 社交广场（私密小组 / 匿名榜单，一键全局关闭）

4. 情侣空间（绑定后可见，单身用户隐藏）
设计逻辑：追求纯净职场用户可完全隐藏所有社交、AI 拓展模块，界面等同于极简待办。

## 5\.3 框架层

1. 任务创建：语音自动识别分类上色、一键拆分多级子任务、标记 P0\-P3 优先级；

2. 专注流程：任务一键启动番茄，基础白噪音免费，无损音效会员解锁；

3. 社交流程：先创建私密小组（P0 上线），公开赛道榜单 P2 迭代，全程匿名开关；

4. 复盘流程：每月自动生成文字总结，看广告 / 会员解锁精美壁纸分享报告；

5. 情侣流程：双人唯一绑定、共享清单、共同目标看板、双人复盘。

## 5\.4 表现层

默认极简黑白低干扰 UI，支持用户自定义任务色彩（红 DDL / 紫工作 / 粉娱乐）；
锁屏 / 桌面小组件轻量化展示当日待办，无多余冗余信息；
复盘报告分职场 / 治愈 / 学习多套高清主题，无水印为尊享会员专属。
