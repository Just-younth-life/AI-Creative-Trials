# AI Creative Trials
本仓库为个人AI创意试验中心，专注轻量化 AI 工具原创开发与持续迭代。 创意落地：产出各类小型 AI 创意 Demo，包含多智能体协同决策引擎、网页端轻量化 AI 应用、RAG 知识库、提示词工程、长文档解析工具等可运行完整项目； 方案迭代：完整留存产品全流程资料，产品定位、PRD 需求文档、竞品分析、视觉色彩规范、交互设计、前端可部署代码，支持持续优化迭代； 思考沉淀：记录 AI 产品落地、多智能体架构、用户交互、人机决策、幻觉治理等方向原创思考，梳理行业痛点、产品差异化、商业化落地逻辑。 所有项目低门槛可复现，兼顾学生学习、产品实践、创业原型验证。
